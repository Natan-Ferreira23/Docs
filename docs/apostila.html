Introdução ao HTML

O HTML é um conjunto de tags responsáveis pela marcação do conteúdo de uma página no navegador.

O HTML disponibiliza um conjunto de tags, exemplo <NomeTag> conteúdo </NomeTag>, colocamos uma tag que representa sua função e o conteúdo no meio da tag de abertura e fechamento.

Por isto o HTML é uma linguagem de marcação e quanto mais marcamos de maneira correta o conteúdo,mais o navegador entende e entrega com prioridade nosso site para o úsuario.

Inicio de uma página HTML ?

Uma página HTML tem seus elementos básicos para funcionar:

  • Html
  • Head
  • Title
  • Body

A tag html identifica o documento como uma pagina de html, ou seja ele é a primeira tag que informamos, todos os comandos ficam dentro dele <html> </html> .

A tag Head contem os meta dados, no caso esses dados não são visiveis ao úsuario, mas são usados pelo navegador <head> </head> .

A tag title fornece o titulo para uma pagina no navegador <title> titulo </title> .

A tag body define o contúdo total e visivel em uma página, dentro dela fica as tags de marcação e os dados que o úsuario interage.

Titulos e parágrafos

Sem dúvidas as tags que mais utilizamos são as tags de Titulo e de parágrafo.

As tags de títulos são h1, h2, h3, h4, h5, h6. Ou seja colocamos a tag <h1> Conteúdo </h1> e colocamos o titulo no meio das tags.

A tag h1 é o titulo mais importante e principal da pagina, e a tag h6 é o menos importante, essa prioridade faz com que o seu site seja entregue mais facilmente para o usuário se bem usado.

As tags de paragrafos são literalmente para paragrafos, escrevemos as informações e textos dentro e é representada por <p> conteudo </p>

Algumas tags de texto

A tag del mostra o contúdo escrito com um risco exemplo: html. Indica que o texto foi deletado, mas ainda sim deixando claro para o usuario.

A tag Sub e Sup: exemplo da tag sup Ab e exemplo da tag sub Ab. A tag sup coloca o conteúdo elevado e a tag sub coloca o conteúdo abaixo.

Tag big e small , exemplos: big small, a tag big é utilizada para aumentar o tamanho da letra em relação ao padrão, e a small é utilizada para diminiuir a letra em relação ao padrão.

A tag cite, de citação é utilizada quando precisamos citar alguma obra literaria, por exemplo um filme ou livro, exemplo: <p> O livro <cite> O banquete </cite>, de Platão. </p>

Tag , strong é utilizada para dar enfase a uma palavra ou frase, o navegador entende que a palavra possui importância quando utilizamos strong. exemplo:<strong> conteudo </strong>

tag em é semelhante a tag strong, da enfase a uma parte, porém não significa que essa parte é mais importante. <em> conteudo </em>.

A tag blockquote, é utilizada quando precisamos trazer uma citação por exemplo do autor/personagem do livro. <blockquote> Só sei que nada sei. </blockquote> Sócrates.

Atributo id e atributo class

Atributo ID:

Um id é um nome dado a certa tag pode ser qualquer tag de abertura, ele identifica de forma única essa tag.

Um id é bem util quando utilizamos CSS, quando precisamos estilizar certa tag e não queremos selecionar todas, selecionamos apenas a que possui id.

Exemplo: <h1 id="titulo"> Conteúdo </h1>

Temos uma tag de titulo com o id titulo, ou seja se quisermos estilizar apenas este conteudo dentro da tag h1, pegamos o id titulo e vamos no CSS e fazemos a estilização referente a ele.

Agora vou fazer outra aplicação ao id.

Imagine que temos uma tag link ao topo da pagina. <a> href="#titulo"> titulo </a>

Repare que no lugar de link temos um #titulo, significa que se apertarmos no link, ele nos levara para a parte da página em que possui um id="titulo".

Ou seja podemos utilizar o link com ID para mover-se pelas partes da mesma página.

Atributo Class:

O atributo class também é bastante utilizado pelo CSS.

Com o atributo class, colocamos classes em uma ou várias tag, essas classes nada mais é que um nome para se referir a elas.

Quando formos estilizar podemos editar todas as tags que colocamos a classe.

Exemplo: <p class="corAmarela"> esta é uma tag de paragrafo </p>

A tag de paragrafo acima tem uma classe chamada corAmarela, ou seja poderiamos colocar em amarelo todos os paragrafos que tivesses essa classe.

Detalhe os ID e Classes, podem ter qualquer nome, mas faz sentido colocar um nome que tenha relação com o uso.

Introdução ao CSS

O CSS oferece uma série de recursos poderosos, como seletores, herança, classes e id, que permitem uma personalização detalhada da aparência de uma página da web.

O CSS é fundamental para uma pagina Web, pois é nele que vamos estilizar tudo, dar cores, formatar tamanhos, controlar posições e etc.

O CSS nos permite abrir um documento em CSS separado para estilizarmos ou simplismente escrever o atributo style e editar na pagina mesmo.

Os formulários permitem o usuario inserir, enviar e interagir com os dados.

Por exemplo para se cadastrarmos em algum site, estamos literalmente preenchendo um formulário.

É um dos recursos mais importantes:

form, a tag form é a primeira de todas, é a tag que agrupa todos os dados de entrada deste formulario, todos os dados que estão dentro de um memso forms vão se relacionar com o mesmo usuario que digitou os dados.

depois temos a tag input ela é a tag que cria uma caixa branca para o usuário digitar seus dados de entrada no formulário.

Note qque dentro dela temos alguns atributos:

  • type
  • name="nome"
  • id="nome"
  • required
  • placeholder

type: significa o tipo do dado de entrada, no exemplo utilizamos text, ou seja um dado tipo texto, pode ser nome, sobrenome ou alguma palavra.

Podemos ter outros tipos de input, como password (senha), E-mail, Number(numeros), note temos os tipos de entradas corretos para cada um.

Exemplos: type="password"; atributo para campos de senha type="email"; atributo de email type="submit"; botão de enviar do formulario

Name, este atributo serve para identificarmos o input de entrada, no caso do exemplo name="nome", sabemos que esse atributo vai receber os nomes do usuario, assim será enviado de forma identificada e referenciada para o servidor. .

ID para cada input dei um id, algo que identifica o campo de entrada de forma única, e note que na tag label coloquei for="id do Input" para que quando clicamos no texto label o cursor no campo input já começa a piscar, sendo intuitivo para o usuario que o label pertence ao seu campo input, e também permite estilizações de forma separada.

Required significa que todo o campo input que tiver required, requer que seja preenchido para o formulário ser enviado, somos obrigados a preencher para poder enviar.

Placeholder serve para tornar mais intuitivo para o usuário no campo em que digitamos os dados, em um input de tipo nome por exemplo, colocamos placeholder="Digite seu nome ", vai aparecer "digite seu nome" no campo input de nome.
